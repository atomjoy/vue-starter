## Vue

Zainstaluj node.js

### Instaluj, kompiluj

```sh
npm install
npm run build
```

## Laravel

Zainstaluj composer

### Aktualizuj

```sh
composer clear-cache
composer update
composer dump-autoload -o
```

### Uruchom

```sh
php artisan serve
php artisan serve --host=localhost --port=8000
```
